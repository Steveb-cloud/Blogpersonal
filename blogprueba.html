Sistema de Gestión de Usuarios

Proyecto para gestionar usuarios utilizando Spring Boot y una base de datos en MariaDB. Se enfoca en la autenticación, autorización y el manejo de información de usuarios.

API Hospitalaria

API RESTful para la gestión de un hospital ficticio. Incluye funcionalidades para manejar pacientes, doctores, citas médicas y el historial médico de los pacientes, utilizando .NET Core y SQL Server.

Tecnologías utilizadas: .NET Core, SQL Server, Entity Framework

Proyecto de Encriptación

Desarrollo de un sistema para la encriptación y desencriptación de datos sensibles. El objetivo es garantizar la seguridad de la información a través de algoritmos de cifrado.
